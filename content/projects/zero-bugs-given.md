## Overview

Zero Bugs Given is a roguelite tower defense game built in Unity. Defend your outposts on a procedurally generated hexagonal map, place turrets, and navigate a risk-reward choice system of bonuses and maluses that shapes every run.

## Key Features

- Hexagonal grid with procedural map generation
- Multiple turret types with unique abilities
- Roguelite bonus/malus choice system
- Boss encounters
